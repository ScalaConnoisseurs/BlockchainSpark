# PoliceSpark

Large scale data processing of police crime statistics.

## Setup

1. Install Java.
2. Install SBT - http://www.scala-sbt.org/download.html
3. Install an IDE (eg. IntelliJ IDEA with Scala plugin).

## Build

1. Clone the repository.
2. cd PoliceSpark
3. sbt compile
